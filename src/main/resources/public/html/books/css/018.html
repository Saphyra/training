CSS - Kiválasztók 2.2: Tartalom kiválasztók

A tartalom kiválasztók tovább segítik a megfelelő elemek kiválasztását, valamint további lehetőségeket adnak a tárolók tartalmának formázásához.

Első / utolsó gyermek

div span:first-child - a <div>-ekben található <span>, ha az a tárolójuk legelső eleme.

div span:last-child - a <div>-ekben található <span>, ha az a tárolójuk legutolsó eleme.

div p:first-child{
    color: red;
}

div p:last-child{
    color: blue
}

Első / utolsó a típusából

Ezek a kiválasztók hasonlóképpen működnek, mint a first-child és last-child, de az azonos típusú tagekből választja ki a legelső / legutolsó elemet.

  • div:first-of-type - a "legidősebb testvér" <div>-ek. (Azok a <div>-ek, amelyek az azonos típusú testvéreik közül a legelsőek.)
  • last-of-type - a "legfiatalabb testvér" tagek. (Azok a tagek, amelyek az azonos típusú testvéreik közül a legutolsóak.)
A legutóbbi példát a következőképpen módosítva: 
div p:first-of-type{
    color: red;
}

div :last-of-type{
    color: blue
}

<DIV>
    <span>Első bekezdés</span>
    <P>Második bekezdés</P>
    <P>Harmadik bekezdés</P>
    <span>Negyedik bekezdés</span>
</DIV>

Magyarázat:
A div p:first-of-type kiválasztó kiválasztja azokat a <p> tageket, amik <div>-en belül vannak, és az azonos típusú testvéreik közül a legelsőek. Ez jelen esetben csak a "Második bekezdés" szöveget tartalmazó <p> tag, mert a tároló <div>-ben összesen van két <p> tag, és ebből a :first-of-type kiválasztó a legelső testvérre szűkíti a találatok listáját.
A div :last-of-type kiválasztó ezzel szemben egyaránt hat a "Negyedik bekezdés"-re és a "Harmadik bekezdés"-re is, mert nem adtuk meg, hogy milyen típusú tageket válasszon ki. "A <div>-en belüli azonos típusú tageknek a legutolsó tagja. A <div>-en belül van 2 <p> és két <span> tag, és mindkét tag csoportból azt az egy taget választotta ki, amely később helyezkedett el az oldalon.

További gyermek kiválasztók

Az :only-of-type azokat az elemeket választja ki, melynek nincs azonos típusú testvére.

Az :only-child azokat az elemeket választja ki, melynek nincs testvére.

Az :empty azokat az elemeket választja ki, melyek üresek.

Példák: 
div :empty{
    width: 3rem;
    height: 1rem;
    background: green;
}

div :only-child{
    color: red;
}

div :only-of-type{
    border: 2px solid black;
}

<DIV>
    <DIV></DIV>

    <DIV>
        <SPAN>DIV-be foglalt SPAN</SPAN>
    </DIV>

    <P>Bekezdés</P>
</DIV>
DIV-be foglalt SPAN

Bekezdés

Jól látszik, hogy a CSS az üres <div>-nek átállította a méretét, és háttérszínét.

A <span> szövegének színe piros lett, mert nincs testvére, így az :only-child kiválasztó alatt megadott szabályok érvényre jutottak.

Az :only-of-type kiválasztó alatti szabályok a <span>-ra, és a <p>-re is érvényre jutottak, mert egyik tagnek sincs azonos típusú testvére.

n-edik elem

Egy tároló meghatározott sorszámú gyermekét a :nth-child(n) kiválasztóval tudjuk kiválasztani, ahol n a gyermek sorszáma.

Példa: 
div :nth-child(3){
    color: red;
}

Ha hátulró szeretnénk számolni, akkor használjuk a :nth-last-child(n) kiválasztót!

Hasonló logika alapján működik az :nth-of-type(n) és :nth-last-of-type(n) kiválasztó, amely az azonos típusú elemek csoportosított listájából választja ki az elölről / hátulról n-edik elemet, típusonként.

Páros / páratlan sorszámú gyermekek kiválasztása

Gondolom, találkoztál már zebra csíkos táblázattal, ahol minden második sornak eltérő háttérszíne volt. Természetesen ezt is CSS-el valósították meg, még pedig a fent bemutatott :nth-child(n) kiválasztóval. A trükk annyi, hogy n helyére nem egy számot, hanem egy függvényt írtak.

Például: tr:nth-child(2n){background-color: green} - minden második sor háttérszínét zöldre állítja.

A fentinél kicsit bonyolultabb függvényeket is meg lehet adni: :nth-child(3n + 2). A CSS úgy dolgozik, hogy az n helyére elkezd számokat behelyettesíteni egyesével léptetve, majd kiszámolja a beírt függvény eredményét, és a kijött eredmény alapján megkeresi az adott sorszámú elemet. Az előbbi függvény alapján a második, majd ötödik, majd nyolcadik, és így tovább... elem tulajdonságait fogja megváltoztatni.

Példa: 
tr:nth-child(2n){
    color: red;
}

tr:nth-child(2n + 1){
    color: blue;
}

tr:nth-child(3n + 2){
    background-color: lightgreen;
}

Megjegyzés: a páros sorokat így is ki lehet vélasztani: :nth-child(odd), míg a páratlanokat így: :nth-child(even)

Tagadó kiválasztás

Vannak esetek, amikor egyszerűbb inkább azt megmondani, hogy mely tageket NEM szeretnénk kiválasztani, ahelyett, hogy melyeket igen.

Ezt a :not(kivalaszto) kombinációval tehetjük meg, ahol azok a tagek lesznek kiválasztva, melyek kivalaszto-nak nem felelnek meg.

Példa: 
div :not(p){
    color: red;
}

    <P>Bekezdés 2</P>
    <SPAN>Nem vagyok bekezdés, SPAN vagyok</SPAN>
    <P>Bekezdés 3</P>
</DIV>
