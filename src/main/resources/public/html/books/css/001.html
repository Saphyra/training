CSS - Bevezetés

Mi az a CSS?

  • A Cascading Style Sheets rövidítése
  • A HTML oldalak megjelenését leíró programozási nyelv

CSS beépítése a weboldalba

CSS kódot 3 módon lehet hozzáadni a HTML oldalhoz:

  1. HTML tag attribútumaként:
    <tag style="CSS kod helye">
    Előnyei:
    • Egyszerű
    • (Majdnem) mindig érvényre jut (lásd később)
    Hátrányai:
    • Csak egyetlen tagre van hatása
    • Hosszú CSS kód olvashatatlanná teszi a HTML-t
  2. Oldal fejlécében (<head>) 
    <head>
    <style>
        CSS kod helye
    </style>
</head>
    Előnyei:
    • Biztosítja a CSS összes lehetőségét
    • Egy szabályt több tagre is lehet alkalmazni
    Hátrányai:
    • Csak az adott oldalra érvényes
    • Hosszú CSS kód olvashatatlanná teszi a HTML-t
  3. Külső fájlban
    1. Egy .css fájl tartalmazza a CSS szabályokat
    2. Ezt a fájlt a weboldal fejlécében beimportáljuk a <link rel="stylesheet" href="css_fajl.css"> tag segítségével.
    Előnyei:
    • A CSS szabályok elkülönülnek a HTML-től
    • Ugyanazt a .css fájlt több oldal is használhatja
    Hátrányai:
    • Ha a CSS fájl valamilyen okból nem elérhető, akkor az oldal rosszul jelenik meg

Versenyhelyzetek

Egy weblap több helyen is tartalmazhat CSS kódot: tag attribútumaként, az oldal fejlécében elhelyezett blokkokban és importált külső fájlokban.

Vegyünk egy egyszerű példát:
Van egy oldalunk, aminek a háttérszínét szeretnénk módosítani. A <body> tag tulajdonságában megadjuk, hogy a háttér színe legyen zöld, a fejlécben azt mondjuk, hogy legyen sárga, míg a beimportált css fájl inkább kékre állítaná.
Na de akkor mi lesz a végeredmény?

A fentebb említett versenyhelyzetekben a következő szerint lépnek érvénybe a szabályok:

  1. A !important (fontosnak jelölt) szabályok
  2. A tag attribútumában megadott szabályok
  3. Amelyik később töltődött be (az importálás és fejlécbe helyezett blokkok sorrendje szerint a legutolsó)
  4. Böngésző beépített szabályai (User Agent Stylesheet)

Szintaktika

kivalaszto {
    tulajdonsag1: ertek1;
    tulajdonsag2: ertek2;
}

A kiválasztókról és tulajdonságokról a későbbi fejezetekben lesz szó.

Példa: 

body {
    background-color: red;
    font-size: 20px;
}
(A <body> háttérszínét pirosra, a betűméretét pedig 20 pixelesre állítja.)

Kommentek: A kommenteket a böngésző figyelmen kívül hagyja a megjelenítés során.
Komment blokk létrehozása: /*Komment szövege*/
Komment blokk lehet több soros, is, de nem ágyazódhatnak egymásba.
