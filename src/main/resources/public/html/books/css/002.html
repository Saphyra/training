CSS - Kiválasztók

Mik azok a CSS kiválasztók (selectorok)?

A CSS felépítése két fő részből áll:

  1. A (megjelenési) szabályok
  2. Azok a kód részletek, melyek megmondják, mely HTML elemekre legyenek érvényesek az egyes szabályok
Ez utóbbit hívjuk kiválasztóknak, mert "Kiválasztják azokat az elemeket (HTML tageket), amelyekre az alattuk felsorolt szabályok érvényre jutnak".

Kiválasztók típusai

  • Mindent kiválasztó kiválasztó

    A * jel minden taget kiválaszt.

  • HTML tag kiválasztása

    Ha taget akarunk kiválasztani, akkor egyszerűen írjuk a kiválasztó helyére a HTML tag nevét.
    Például:
    i {}: Kiválasztja az összes <i> taget.

  • ID kiválasztása

    ID (azonosító) kiválasztásához két dolgot kell tenni:
    1. A kiválasztandó HTML taget fel kell ruházni az id attribútummal:
      <b id="egyedi_azonosito">
    2. Meg kell írni a CSS kiválasztót, ami #-el kezdődik, és a kiválasztandó HTML tag id attribútumának értékével folytatódik (#egyedi_azonosito)
      Ez a kiválasztó (szabvány szerint) legfeljebb egy taget választhat ki.
    Megjegyzés: szabvány szerint minden HTML tag id attribútumának különbözőnek kell lenni (ha van), ezért egyedi azonosító. Ellenben a gyakorlat azt mutatja, hogy (egyes böngészők esetében) a CSS szempontjából az ID-k egyedisége nem befolyásolja a működést.

  • Osztály kiválasztása

    Osztály kiválasztásakor a teendő hasonló, mint az id kiválasztásakor, először is a megfelelő HTML tageket kell felruházni ez esetben a class attribútummal:
    <sup class='osztalynev'>
    <del class='osztalynev'>
    Mint látható, azonos osztály név többször is előfordulhat a dokumentumban, és a tageknek sem kell azonosnak lenniük.
    A selector is hasonló felépítésű, mint az id kiválasztásánál, csak itt a # helyére egy . (pont) kerül, majd utána a kiválasztandó osztály neve:
    .osztalynev: kiválasztja az összes olyan taget, amelynek az osztálya osztalynev
    Megjegyzés: egy tag több osztálynak is tagja lehet. Ilyenkor a class attribútumba szóközökkel elválasztva kell felsorolni az összes osztályt, melynek tagja az adott tag.
    Például: <ins class="osztaly1 osztaly2 osztaly3">
