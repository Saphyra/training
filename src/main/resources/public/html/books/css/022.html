CSS - Reszponzivitás

Mi az a reszponzivitás?

Valószínűleg minden ember találkozott már olyan oldallal, ami telefonon másképpen jelent meg, mint egy számítógép kijelzőjén. Ennek az oka a következő: Egy telefon általában sokkal nagyobb képpont sűrűséggel rendelkezik, mint egy asztali monitor (telefon: 2000 képpont 15-20 centire sűrítve, mig egy monitoron ugyanennyi képpontra 30-50 centi is jut, ha nem több). Ez azzal jár, hogy a telefonokon a 16 pixeles betűméret sokkal kisebbként jelenik meg, mint az asztali gépek esetében. Másik példa, míg a telefonokat legtöbbször állított helyzetben bámuljuk, addig asztali gépek szinte mindig fektetve vannak. Így könnyen előfordulhat, hogy ami a számítógépünkön szépen és olvashatóan jelenik meg, ugyanaz a weboldal telefonon el van csúszva, nem fér ki, vagy éppen olvashatatlanul kicsi a szöveg.

Ha egy weboldal érzékeli a kijelző paramétereit, és ennek függvényében jeleníti meg a tartalmat, akkor azt reszponzív weboldalnak tekinthetjük.

Hogyan legyünk reszponzívak?

Alap szintű reszponzivitást a legkönnyebb relatív mértékegységek használatával, és a tárolók méretének minimum és maximum értékeinek meghatározásával elérni. Természetesen a CSS ennél sokkal kifinomultabb és precízebb megoldásokat is nyújt. Ezek közül az egyiket media query-nek hívjuk.

media queryk

A media queryk olyan szabályokat foglalnak magukban, melyek csak akkor jutnak érvényre, hogyha a képernyő bizonyos feltételeknek megfelel.

Szintaktika: @media not/only megjelenito and (feltetel) and (feltetel) ... { szabályok }
  • @media: media query létrehozása
  • not/only megjelenito: Szűrés kijelzőtípusra.
    Az első szó a szűrés típusát adja meg:
    • not: minden kijelzőtípus, kivéve a paraméterként megadott
    • only: csak a paraméterként megadott kijelzőtípus
    A második szó a kijelző típusát adja meg, ami lehet:
    • print: nyomtató
    • screen: kijelző
    • speech: felolvasó program (látássérülteknek például)
  • and (feltetel): további feltételek
Ha a megadott feltételek mindegyike teljesül, akkor érvényre jutnak a media queryben foglalt szabályok.
Példa: 
@media only screen and (min-width: 600px) and (max-width: 1000px) {
    body{
        background-color: red;
    }
}
Ha a megjelenítő típusa kijelző (képernyők, monitorok...), és az minimum 600, maximum 1000 pixel széles, akkor a háttér színét pirosra állítja.
