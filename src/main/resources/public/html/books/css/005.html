CSS - Szegélyek

Szegély megadása

Szegélyt a borderrel kezdődő tulajdonságokkal lehet létrehozni.

A tulajdonság 3 értéket vár:

  1. A szegély vastagsága
  2. A szegély típusa (Lásd: Szegélytípusok)
  3. A szegély színe
Például: border: 3px solid red; - létrehoz egy 3 pixel vastag, pirols, folyamatos vonalból álló szegélyt a tároló köré. Példa

Szegély megadása a tároló bizonyos oldalaira

A CSS lehetőséget ad a tároló oldalainak szegélyeit külön-külön beállítani.

Ezt ugyanúgy lehet megtenni, mint a körkörös szegélynél, csak a tulajdonság végéhez hozzá fűzzük, hogy melyik oldal szegélyét szeretnénk állítani (top: felső, right: jobb oldal, bottom: alsó, left: bal oldal).
Például: border-bottom: 3px solid red; - Az előző szegély csak az alsó oldalra alkalmazva. Példa

A szegély tulajdonságainak külön történő megadása

A szegély tulajdonságit meg lehet adni külön-külön is, például border-color: red;, vagy csak az adott oldalra: border-bottom-color: red;
Az így beállítható tulajdonságok:

  • border-color: szegély szín
  • border-style: szegély stílus
  • border-width: szegély vastagság

Szegély típusok

  • dotted: pontozott
  • dashed: szaggatott vonal
  • solid: egyszerű vonal
  • double: dupla vonal
  • groove: groove
  • ridge: ridge
  • inset: inset
  • outset: outset
  • none: nincs szegély
  • hidden: láthatatlan szegély

Lekerekítés

A szegély lekerekítését a border-radius tulajdonsággal lehet állítani.

border-radius: 15px;
Példa

Sarkonként különböző lekerekítés megadása: A border-radius tulajdonságnak 4 értéket adunk egymás után, a bal felső sarokból indulva az óramutató járásának megfelelően:
border-radius: 10px 15px 20px 25px;

Ha csak egy saroknak szeretnénk módosítani a lekerekítését, akkor azt a sarok megnevezésével is megtehetjük.
Például a bal alsó sarok szegélyét a border-bottom-left-radius tulajdonsággal állíthatjuk.

Megjegyzés: a border-radius tulajdonság használható bármilyen tároló lekerekítésére, akár szegély nélkül is. Állítsuk az értékét 50%-ra a teljesen kerek forma eléréséhez!
