CSS - Kiválasztók 2.1: Állapot kiválasztók

Állapot kiválasztókról általában

A HTML tageknek lehetnek bizonyos, felhasználói beavatkozástól függő állapotai. És elég gyakran előfordul, hogy bizonyos szabályokat csak akkor szeretnénk alkalmazni, ha ezek a z állapotok fennállnak.

Szintaktika: kivalaszto:allapotkivalaszto

Linkek állapotai

Bizonyára feltűnt neked is, hogy a Google keresőjében (és sok más helyen) azok a linkek, amelyeket már meglátogattál, másképp néznek ki. Ezt a hatás nagyon jó példa az állapot kiválasztók működésére.

Amikor egy linkre rákattintanak, akkor annak az állapota megváltozik, és ha van ezt a változást figyelő CSS kód (alapértelmezetten van), akkor az az állapotváltozás következtében érvényre jut, és megváltoztatja a link tulajdonságait (alapértelmezetten a színét).

Linkek állapotai, és kiválasztói:
  • :link - az a link, amelyet még nem látogattak meg
  • :visited - meglátogatott link
  • :active - az a link, amelyre éppen rákattintottak (kattintás effekt)
Használata: a:visited {}
Példa: 
a:link{
    color: red;
}
a:visited{
    color: yellow;
}
a:active{
    font-size: 4rem;
}
Kattints rám, és figyeld, mi történik!

Hover effect

A :hover állapot kiválasztó abban az esetben jut érvényre, hogyha az egér a kiválasztott elem fölé kerül.

Ezzel nagyon látványos és hasznos trükköket lehet csinálni: 
#hover_test_hidden{
    display:none;
}
#hover_test span:hover + div{
    display: block;
}

<DIV id="hover_test">
    <SPAN style="border: 1px solid red">Húzd fölém az egeret!</SPAN>
    <DIV id="hover_test_hidden">Azt hitted, hogy nem vagyok itt, hahaha!</DIV>
</DIV>
Húzd fölém az egeret!
Azt hitted, hogy nem vagyok itt, hahaha!

Magyarázat: a hover_test_hidden azonosítójú elemet elrejtjük a képernyőről. Viszont a #hover_test span:hover + div kiválasztó kombinációval (válaszd ki azt a <div> taget, amely KÖZVETLENÜL egy olyan <span> mögött helyezkedik el, amely hover_test azonosítújú elemen belül helyezkedik el, és éppen fölötte van az egér) megjelenítjük, de csak abban az esetben, ha a megfelelő elem fölött van az egér.
