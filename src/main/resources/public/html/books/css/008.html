CSS - Margók

Mire jók a margók?

Margók beállításával a tárolóink egymáshoz képesti helyzetét tudjuk meghatározni.

Például vegyünk egy nagy dobozt, amibe bele rakunk egy kisebbet úgy, hogy annak egy meghatározott távolságra kell lennie a nagy doboz sarkától. Ezt a távolságot adja meg a margó:
margin: 20px;: Egy 20 px széles területet foglal le a tároló körül. 

<div style='background: red'>
    Tároló
    <div style='background: green; margin: 20px'>Doboz margóval</div>
    <div style='background: yellow;'>Margó nélküli doboz</div>

</div>
Tároló
Doboz margóval
Margó nélküli doboz

Ahogy a példában látható, a margóval ellátott doboz körül létrejött egy 20 pixel vastag üres tér.
