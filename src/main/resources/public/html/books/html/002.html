HTML - Weboldal készítése

Szerkesztő program

HTML kódot gyakorlatilag bármilyen szövegszerkesztő programmal lehet írni, habár vannak olyan szerkesztők is, melyek segítik a fejlesztőt a gyorsabb kódolásban:

  • Notepad++ ingyenes, egyszerű, gyors, és számos programozási nyelvhez nyújt támogatást.
  • A Brackets egy ingyenes IDE (Integrált fejlesztői környezet) elsősorban weboldalak fejlesztéséhez kialakítva.

Készítsük el első HTML oldalunk!

  1. Hozzunk létre egy új fájlt elso.html néven!
  2. Nyissuk meg ezt a fájlt az általunk választott szövegszerkesztő programban! (Én a Notepad++-t fogom használni)
  3. Adjuk meg a dookumentum típusát!
  4. Hozzuk létre az oldal alap struktúráját!
    Minden HTML oldalnak egyetlen gyökere lehet, ami a <html> tag.
    Ebbe kerül a fejléc (<head>) és a törzs (<body>).
    Ne felejtsük el a tageket a megfelelő helyen lezárni! Megjegyzés: Érdemes a tagek gyermekeit egy kicsit beljebb tolni, így a kód sokkal olvashatóbbá válik. Az extra szóközök, vagy tabulátorok amúgy sem jelennek meg a böngészőben.
  5. Az így létrehozott oldalunkat töltsük fel tartalommal!
    Adjunk neki egy címet (<title>), készítsünk egy főcímet (<h1>), és egy bekezdést (<p>)!
  6. Legvégül pedig nyissuk meg az első weblapunkat egy böngészőben! (Dupla klikk a fájlra, vagy jobb klikk -> társítás -> és válasszuk ki a böngészőt.) Ahogy a képen is látszik, az általunk beírt tagek nem jelentek meg a képernyőn, viszont megváltoztatták a tartalom kinézetét. (A böngészőfülben megjelent az oldal címe, a címsor pedig nagyobb lett.)
